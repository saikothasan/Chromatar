### API Usage

Generate avatars programmatically using the API:

```
https://chromatar.pages.dev/api/avatar?text=V0&gradient=1&size=400
```

Parameters:
- `text`: 1-2 characters (required)
- `gradient`: 1-6 (default: 1)
- `size`: 64, 128, 256, or 400 (default: 400)
